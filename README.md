# Computer Shop

An online computer shop with a PC Builder featuring a full compatibility rules engine.

## Structure

```
cmp-shp/
├── frontend/   # Next.js + Tailwind CSS
└── backend/    # API server + Supabase integration
```

## Features

- Online store with product variants
- PC Builder with full hardware compatibility engine
- Flexible Excel importer for bulk product migration
- Admin panel for product management
